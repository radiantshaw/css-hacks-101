Specified vs Unspecified

Specified properties


              p {
                color: salmon;
              }

Unspecified properties


              p { /* ... */ }

Calculation rules

  • For non-inherited properties, the initial value is used on all elements, as long as no specified value is supplied.
  • For inherited properties, the initial value is used on the root element only, as long as no specified value is supplied.

z-index

Default stacking order

  1. Root element (<html>)
  2. Non positioned elements in the defined order
  3. Positioned elements in the defined order

              

                

              

              

              

            

              .blue, .pink, .orange { position: absolute }

Stacking using z-index


              .blue, .pink, .orange { position: absolute }
              .blue { z-index: 2 }
              .orange { z-index: 3 }
              .green { z-index: 100 }

Stacking context


              

                

              

              

              

              

            

              .blue, .pink, .orange, .purple { position: absolute }
              .purple { z-index: 0 }
              .pink { z-index: 1 }
              .blue { z-index: 2 }
              .orange { z-index: 3 }
              .green { z-index: 100 }
